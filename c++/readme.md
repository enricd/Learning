### C basics

- write a "Hello, World!" program in C++ in simple_c.c
- compile it with gcc -o simple_c simple_c.c
- run it with ./simple_c

### random C with custom library

- create a random_c.c that uses functions from random.h and random.c
- compile it with gcc -o random_c random_c.c random.c -lm
- run it with ./random_c
